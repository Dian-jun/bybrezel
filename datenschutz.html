개인정보처리방침

Brezel Money는 사용자의 금융 데이터를 소중히 여깁니다.

1. 운영자 정보

서비스명: Brezel Money

웹사이트: bybrezel.com/money

문의: bybrezel.com 내 연락처 참조

2. 수집하는 데이터

✓ Brezel Money는 서버에 개인 금융 데이터를 저장하지 않습니다. 모든 가계부 데이터는 사용자의 기기(브라우저 localStorage)에만 저장됩니다.

수집하거나 처리하는 데이터:

  • 가계부 내역, 계좌 정보, 고정비 — 기기 내 저장만, 서버 전송 없음
  • 자연어 입력 텍스트 — 파싱 목적으로 Anthropic API에 전송 (아래 참조)
  • 환율 정보 — 외부 환율 API에서 조회 (개인정보 미포함)

3. 제3자 서비스

Anthropic API (자연어 입력 기능)

자연어로 내역을 입력할 때 입력한 텍스트가 Anthropic의 서버로 전송됩니다. 이 기능은 선택 사항이며, 직접 입력 방식을 사용하면 데이터가 전송되지 않습니다.

예: "rewe에서 50유로 카드로" → Anthropic API에서 파싱 후 결과 반환

Anthropic의 개인정보처리방침: anthropic.com/privacy

환율 API

현재 환율 조회를 위해 공개 환율 API를 사용합니다. 개인정보는 전송되지 않습니다.

4. 쿠키 및 추적

Brezel Money는 쿠키를 사용하지 않으며 사용자 행동을 추적하지 않습니다. 광고 또는 분석 도구가 포함되어 있지 않습니다.

5. 데이터 보관 및 삭제

모든 데이터는 사용자 기기의 브라우저 localStorage에 저장됩니다. 데이터 삭제 방법:

  • 앱 내 설정 → 앱 초기화 버튼
  • 브라우저 설정에서 사이트 데이터 삭제

6. 사용자 권리 (GDPR)

EU/EEA 거주자로서 다음 권리를 갖습니다:

  • 데이터 열람권 — 모든 데이터는 기기에 저장되어 직접 확인 가능
  • 삭제권 — 앱 초기화로 즉시 삭제 가능
  • 이동권 — 설정 → 데이터 내보내기로 JSON 다운로드 가능

7. 미성년자

Brezel Money는 만 16세 이상을 대상으로 합니다.

8. 방침 변경

방침 변경 시 이 페이지에 업데이트 날짜와 함께 공지합니다.

Datenschutzerklärung

Brezel Money schützt Ihre finanziellen Daten.

1. Verantwortliche Stelle

Dienst: Brezel Money

Website: bybrezel.com/money

Kontakt: Siehe Impressum unter bybrezel.com

2. Erhobene Daten

✓ Brezel Money speichert keine persönlichen Finanzdaten auf Servern. Alle Haushaltsbuchdaten werden ausschließlich lokal auf dem Gerät des Nutzers (Browser localStorage) gespeichert.

Verarbeitete Daten:

  • Buchungseinträge, Kontodaten, Fixkosten — nur lokale Speicherung, keine Serverübertragung
  • Freitexteingaben — zur Analyse an die Anthropic API übermittelt (siehe unten)
  • Wechselkursdaten — Abfrage über externe Wechselkurs-API (keine personenbezogenen Daten)

3. Drittanbieter

Anthropic API (Freitexteingabe)

Bei der Freitexteingabe wird der eingegebene Text zur Verarbeitung an Anthropic übermittelt. Diese Funktion ist optional. Bei manueller Eingabe werden keine Daten übertragen.

Beispiel: "50 Euro bei REWE mit Karte" → Verarbeitung durch Anthropic API → Rückgabe des Ergebnisses

Datenschutzerklärung von Anthropic: anthropic.com/privacy

Wechselkurs-API

Zur Abfrage aktueller Wechselkurse wird eine öffentliche API genutzt. Es werden keine personenbezogenen Daten übertragen.

4. Cookies und Tracking

Brezel Money verwendet keine Cookies und verfolgt kein Nutzerverhalten. Es sind keine Werbe- oder Analysetools eingebunden.

5. Datenspeicherung und -löschung

Alle Daten werden im Browser localStorage des Nutzergeräts gespeichert. Möglichkeiten zur Datenlöschung:

  • In der App: Einstellungen → App zurücksetzen
  • Über die Browsereinstellungen: Website-Daten löschen

6. Betroffenenrechte (DSGVO)

Als in der EU/EEA ansässige Person haben Sie folgende Rechte:

  • Auskunftsrecht — Alle Daten sind lokal gespeichert und direkt einsehbar
  • Recht auf Löschung — Sofortige Löschung über App zurücksetzen möglich
  • Recht auf Datenübertragbarkeit — Export als JSON über Einstellungen → Daten exportieren

7. Minderjährige

Brezel Money richtet sich an Personen ab 16 Jahren.

8. Änderungen dieser Erklärung

Bei Änderungen wird diese Seite mit dem aktualisierten Datum aktualisiert.
