Brezel Money의 개선 사항과 수정 내역을 기록합니다.

v0.3 2026년 4월 15일
NEW

회원가입 · 로그인 — 계정을 생성하고 로그인할 수 있는 기능을 추가했어요.

NEW

기기 간 데이터 동기화 — 데이터가 DB에 저장되어 기기를 변경해도 자동으로 동기화돼요.

NEW

데이터 백업 — 기존에 로컬 저장소를 사용하던 사용자도 로그인하면 데이터를 백업하고 이어서 사용할 수 있어요.

개선

환율 기준 표시 — 환율 기준 일자를 명확히 확인할 수 있도록 개선했어요.

개선

환율 정밀도 — 환율을 소수점 두 자리까지 표시할 수 있도록 개선했어요.

개선

환율 적용 방식 — 환율을 하루에 한 번만 불러오도록 변경했어요. 이로 인해 계좌 간 이동 시 시간 차이로 발생하던 미세한 금액 차이가 더 이상 생기지 않아요.

FIX

온보딩 고정 수입 입력 — 항목을 추가하지 않고 바로 저장해도 입력 중이던 고정 수입이 정상적으로 등록되도록 수정했어요.

v0.2 2026년 4월 14일
FIX

온보딩 계좌 등록 — 잔액 입력 시 네 자리 이상 입력이 안 되던 오류를 수정했어요.

FIX

온보딩 현금 잔액 — 현금 보유액 입력 시 네 자리 이상 입력이 안 되던 오류를 수정했어요.

FIX

홈 → 내역 수정 — 홈 화면에서 내역을 수정할 때 앱이 튕기던 오류를 수정했어요.

NEW

내역 필터 — 거래 내역에 정렬(날짜순·금액순 등)과 기간 필터(이번 달·지난 달·직접 입력 등) 기능을 추가했어요.

NEW

송금 기능 — 내역 추가에서 한국 ↔ 독일 계좌 간 송금 유형을 지원해요. 수수료도 함께 기록할 수 있어요.

개선

자산 날짜별 잔액 — 특정 날짜 기준 총 자산 및 계좌별 잔액을 달력으로 조회할 수 있어요.

개선

온보딩 고정 지출 — 온보딩 완료 시 고정 지출 내역에 환율이 올바르게 적용되지 않던 부분을 개선했어요.
